# 笔记
## 写在前面
- 跟随作者使用了torch1.13.1版本，配置其他包来兼容这个版本的torch
- 服务器（没有魔法）不能访问 HuggingFace，导致很多东西无法下载
- 服务器启动服务以后，想要在本地浏览器中打开，需要做 SSH 端口转发，将服务器的端口映射到本地电脑上
    - 服务器上的代码保持默认（不用改 server_name，用 launch() 就行），启动后会在 127.0.0.1:7860 运行。
    - 在你本地电脑的终端里，执行 SSH 端口转发命令：
    ```
    # 把服务器的7860端口，映射到你本地的7860端口
    ssh -L 7860:127.0.0.1:7860 xyh@你的服务器IP
    ```
    - 然后在你本地浏览器里，访问 http://127.0.0.1:7860 就能打开 Gradio 界面了。
